# Neuralnoy Project

A simple monorepo containing:
- `backend/`: A Rust Cargo project
- `frontend/`: A React TypeScript project powered by Vite

## How to run

1. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

2. **Backend**:
   ```bash
   cd backend
   cargo run
   ```

Or from the root directory:
- `npm run install:frontend`
- `npm run dev:frontend`
- `npm run dev:backend`
